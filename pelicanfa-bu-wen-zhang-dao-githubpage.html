    pelican发布文章到GithubPage

    pelican发布文章到GithubPage

    安装

    pip install "pelican[markdown]"

    创建一个项目

    pelican-quickstart

    设置静态文件路径

    在 pelicanconf.py 中设置静态文件路径

    STATIC_PATHS = ['pics', 'images']

    创建一篇文章

    编辑以下文章内容

    Title: 第一篇文章
Date: 2010-12-03 10:20
Category: Review

这是一篇简单的小文章

    保存到项目 content目录下

    ~/projects/yoursite/content/小文章.md

    生成网站

    创建theme目录

    mkdir theme

    下载主题，以 pelican-cait 为例子

    git clone https://github.com/hdra/pelican-cait.git

    编译，并指定主题

    pelican content -s pelicanconf.py -t /path/to/your/theme

    预览

    不指定主题

    pelican --listen

    指定主题

    pelican content -s pelicanconf.py -t /path/to/your/theme

    发布到 Github

    创建 xxx.github.io 项目，在生成的 polican 目录下的 output 目录下初始化 git，

    cd ./output
git init .
git add .
git commit -m "博客"
git remote add origin [xxx你的github.io.git]
git branch -M main
git push -u origin main
